# Introdução à Inteligência Artificial

Este repositório armazena os trabalhos práticos da disciplina de **Introdução à Inteligência Artificial**, sob orientação do professor Li Weigang, referente ao semestre 2025.2.

O objetivo é explorar algoritmos e técnicas fundamentais de IA utilizando Jupyter Notebooks e bibliotecas padrão da área.

## Organização do Repositório:

- **`trabalho1`**: Primeiro trabalho prático, focado em fundamentos e algoritmos de busca ou lógica básica.
- **`trabalho2`**: Segundo trabalho prático, avançando para tópicos como machine learning e heurísticas.
